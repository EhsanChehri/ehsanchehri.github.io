گیت هاب امکانات فوق العاده ای داره و همه برنامه نویسارو شیفته خودش کرده اما واسه منم جالبه، سرعت و کیفیتش مجذوبم کرده

خام بودن و اجبار به یادگیری یکی از رازهای جکیل بوده تا حالا واسه من

آشنا شدن با گیت و گیت هاب خالی از لطف نبود

جکلیل خودش محدودیت های زیادی واسه کسی که از وردپرس مهاجرت کنه داره اما جکلیل روی گیت هاب بازم محدودتر میشه

اولین کاری که واسه بهبود اوضاع انجام دادم درج کنونیکال و تنظیم خطای 404 بود

---

برای افزودن کنونیکال از کدهای آماده [اینجا](https://gist.github.com/bennylope/1894706#gistcomment-2821633) استفاده کنید

```
sudo apt install g++
sudo apt install cmake
sudo apt install cmake-curses-gui
```
